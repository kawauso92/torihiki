# asset-app

暗号資産・入金記録・損益管理アプリ

## 目的
- 入金額を記録する
- 現在価値を表示する
- 損益を表示する
- 期間指定で集計する
- 資産推移をグラフで見る
- 将来的に GMO / MetaMask / Phantom / DEX ポジション取得に対応する

## 開発方針
- まずは Windows 上で動くローカル Web アプリ
- Git / GitHub で変更履歴を管理
- 外部公開はしない
- 秘密鍵やシードフレーズは扱わない
- API キーは .env でローカル管理する
